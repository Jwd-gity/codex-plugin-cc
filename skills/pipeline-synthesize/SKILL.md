# Pipeline Synthesize

## 描述

读取多个阶段的输出文件，整合为结构清晰的最终文档。处理冲突、去重、统一格式。

触发关键词：`synthesize`、`整合`、`汇总`

## 参数

- inputs: string    # 逗号分隔的输入文件路径列表
- output: string    # 最终输出文件路径

## 工具

Read, Write, Edit, Grep, Glob

## 步骤

1. 按顺序读取所有 `inputs` 文件
2. 提取各文件核心内容
3. 检测重复或冲突信息并合并
4. 统一格式和术语
5. 生成最终文档
6. 写入 `output` 文件

## 输出格式

写入 `output` 路径的文件：

```markdown
# [整合标题]

> 整合来源：[列出输入文件]

## 摘要
[一段话概括全貌]

## 详细内容
[按主题组织的完整内容]

## 附录
- 来源文件：[列表]
- 生成时间：[时间戳]
```

## 边界

- 不丢失任何输入数据（可重组，不可丢弃）
- 标注信息来源
- 输出失败时写入部分结果 + 错误说明
